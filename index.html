Awatistic

a fast AWA5.0 interpreter
AWA5.0 is an esoteric programming language made up by the community of Jelly Hoshiumi. The reference implementation is in JavaScript so I implementd my own interpreter in Rust to achieve maximum meme potential. Also works as assembler and disassembler with some debugging capabilities.
